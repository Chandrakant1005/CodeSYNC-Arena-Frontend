# CodeSYNC Arena

Full-stack meeting platform with:

- React frontend
- Node.js and Express backend
- MySQL database
- JWT auth plus session-backed meeting presence
- Join by meeting ID or room URL
- Owner prompts when users join or leave

## Project structure

- `client` - React + Vite frontend
- `server` - Express + MySQL backend

## Features

- Signup and login
- Session-aware protected routes
- Create meeting
- Join meeting by ID or URL
- Meeting owner alerts for join and leave events
- Light-theme meeting UI inspired by the provided reference
